# 사용자 흐름

## 표기 원칙

- `[화면]`은 사용자가 보는 화면, `{판단}`은 분기, `[(로컬)]`은 로컬 저장을 뜻한다.
- OCR 후보는 `검토 완료` 전 단어 엔터티로 저장되지 않는다.
- 취소·외부 오류가 발생해도 가능한 한 작업 draft를 로컬에 남긴다.

## 1. 사진 한 장으로 단어장 생성

```mermaid
flowchart TD
    A[홈] --> B[단어 가져오기]
    B --> C{입력 방식}
    C -->|카메라| D[사진 촬영]
    C -->|앨범| E[앨범에서 선택]
    D --> F[미리보기·자르기]
    E --> F
    F --> G[(원본/작업 로컬 저장)]
    G --> H[OCR 처리 중]
    H --> I{처리 결과}
    I -->|성공| J[OCR 결과 검토]
    I -->|실패| K[OCR 실패]
    K --> F
    J --> L{선택 단어 있음?}
    L -->|아니요| J
    L -->|예| M[검토 완료]
    M --> N[새 단어장 이름·분야]
    N --> O[(단어장과 확정 단어 저장)]
    O --> P[단어장 상세]
```

## 2. 여러 장 연속 업로드

여러 장 업로드는 MVP 후반 범위다. 각 장은 독립 작업 상태를 가지며 한 장 실패가 전체를 취소하지 않는다.

```mermaid
flowchart TD
    A[앨범에서 여러 장 선택] --> B[선택 순서와 장 수 확인]
    B --> C{다음 이미지 있음?}
    C -->|예| D[이미지 미리보기·자르기]
    D --> E[(해당 장 로컬 저장)]
    E --> F[다음 이미지]
    F --> C
    C -->|아니요| G[OCR 작업 큐 시작]
    G --> H{장별 처리}
    H -->|성공| I[후보 저장]
    H -->|실패| J[해당 장 실패 표시]
    I --> K{남은 장 있음?}
    J --> K
    K -->|예| H
    K -->|아니요| L[장별 탭이 있는 OCR 결과 검토]
    L --> M[중복 경고·통합 선택]
    M --> N[검토 완료 후 단어장 선택]
```

## 3. OCR 오인식 수정

```mermaid
flowchart TD
    A[OCR 결과 검토] --> B[확인 필요 후보 선택]
    B --> C[원문 이미지 위치 확대]
    C --> D[편집 열기]
    D --> E[철자 수정]
    E --> F{영어 단어 형식인가?}
    F -->|아니요| G[조건 안내·입력 유지]
    G --> E
    F -->|예| H[정규화·중복 확인]
    H --> I{중복인가?}
    I -->|예| J[중복 합치기/둘 다 유지 선택]
    I -->|아니요| K[후보 draft 갱신]
    J --> K
    K --> L[검토 목록으로 복귀]
    L --> M[검토 완료]
    M --> N[(확정 단어 저장)]
```

## 4. 단어 수동 추가

```mermaid
flowchart TD
    A[단어장 상세 또는 OCR 검토] --> B[직접 단어 추가]
    B --> C[영어 단어 입력]
    C --> D{형식과 중복 확인}
    D -->|오류| E[입력 아래 조건 안내]
    E --> C
    D -->|중복| F[기존 단어 열기/별도 저장]
    D -->|통과| G[뜻·품사 선택 입력]
    F -->|별도 저장| G
    F -->|기존 열기| H[단어 상세]
    G --> I[단어 저장]
    I --> J[(로컬 단어 생성)]
    J --> H
```

## 5. 이미지 후보 선택

```mermaid
flowchart TD
    A[단어 상세] --> B[연상 이미지 선택]
    B --> C[무료 이미지 검색]
    C --> D{결과와 라이선스 있음?}
    D -->|예| E[후보 grid]
    E --> F[이미지 선택]
    F --> G[작가·출처·라이선스 확인]
    G --> H{사용할까?}
    H -->|아니요| E
    H -->|예| I[이 이미지 사용]
    I --> J[(이미지와 출처 메타데이터 저장)]
    J --> K[단어 상세]
    D -->|아니요| L[이미지 검색 실패 흐름]
```

## 6. 이미지 검색 실패

```mermaid
flowchart TD
    A[이미지 검색] --> B{실패 종류}
    B -->|네트워크| C[연결 오류와 로컬 보존 안내]
    B -->|결과 없음| D[알맞은 이미지 없음]
    B -->|라이선스 없음| E[선택할 수 없는 이유 표시]
    C --> F[다시 시도/나중에]
    D --> G[검색어 수정/이미지 없이 계속]
    E --> G
    F -->|다시 시도| A
    G -->|검색어 수정| A
    F -->|나중에| H[단어 상세]
    G -->|이미지 없이| H
    G -. 출시 후 .-> I[AI 이미지 생성 명시적 요청]
    I --> J[AI 생성 표시 후 저장]
```

## 7. 플래시카드 학습

```mermaid
flowchart TD
    A[단어장 상세] --> B[플래시카드 학습 시작]
    B --> C[앞면: 단어/이미지]
    C --> D[답 떠올리기]
    D --> E[답 보기]
    E --> F[뒤면: 뜻·품사·예문·연상]
    F --> G{다음 카드 있음?}
    G -->|예| C
    G -->|아니요| H[학습 완료 요약]
    C --> I{중도 종료?}
    I -->|예| J[(진행 위치 로컬 저장)]
    J --> K[단어장 상세]
    I -->|아니요| D
    H --> K
```

## 8. 오답 단어 복습

```mermaid
flowchart TD
    A[오늘의 복습] --> B[복습 시작]
    B --> C[카드 앞면]
    C --> D[답 보기]
    D --> E{기억 평가}
    E -->|다시| F[짧은 간격으로 재학습 큐]
    E -->|어려움| G[짧은 다음 일정]
    E -->|보통| H[기본 다음 일정]
    E -->|쉬움| I[긴 다음 일정]
    F --> J[(기록+일정 원자적 저장)]
    G --> J
    H --> J
    I --> J
    J --> K{남은 카드?}
    K -->|예| C
    K -->|아니요| L{재학습 카드?}
    L -->|예| C
    L -->|아니요| M[복습 완료 요약]
    M --> N[통계/학습 홈]
```

## 9. 첫 실행 권한 요청

```mermaid
flowchart TD
    A[첫 실행 온보딩] --> B[홈]
    B --> C[단어 가져오기]
    C --> D{사용자가 선택한 방식}
    D -->|사진 촬영| E[카메라 사전 안내]
    D -->|앨범| F[앨범 사전 안내]
    E --> G[시스템 카메라 권한 요청]
    F --> H[시스템 사진 선택/권한]
    G --> I{허용?}
    H --> J{선택/허용?}
    I -->|예| K[사진 촬영]
    I -->|아니요| L[권한 거부 화면]
    J -->|예| M[앨범 업로드]
    J -->|취소/거부| N[이전 화면]
    L --> O[설정 열기/앨범 대안]
    O -->|설정 후 허용| K
    O -->|앨범| M
```

## 10. 앱 재실행 후 학습 이어하기

```mermaid
flowchart TD
    A[앱 재실행] --> B[로컬 작업·세션 조회]
    B --> C{미완료 항목}
    C -->|OCR 처리/검토| D[홈: 검토 이어하기 banner]
    C -->|플래시카드 세션| E[홈/학습: 학습 이어하기]
    C -->|없음| F[일반 홈]
    D --> G{사용자 선택}
    G -->|이어하기| H[OCR 결과 검토 또는 처리 상태]
    G -->|나중에| F
    E --> I{사용자 선택}
    I -->|이어하기| J[저장된 다음 카드]
    I -->|처음부터| K[세션 재시작 확인]
    J --> L[(새 진행 위치 저장)]
    K --> J
```
