# 제품 정의

## 사용자 문제

종이 교재나 인쇄된 단어 목록으로 공부하는 사용자는 단어를 다시 입력하고, 뜻·발음·예문을 여러 곳에서 찾고, 복습 카드를 만드는 데 학습보다 많은 시간을 쓴다. 기존 OCR은 오인식을 피하기 어렵고, 이미지 단어장은 적절한 이미지를 찾거나 출처를 관리하기 어렵다. 특히 수능·토익·간호·의학 영어는 문맥에 따라 뜻과 품사가 달라 일반 사전 결과만으로 부족하다.

LinkVoca는 OCR을 “정답 자동 생성”이 아니라 “입력 보조”로 사용한다. 사용자가 추출 결과를 확정한 뒤에만 단어장에 저장하고, 이미지 연상과 간격 반복으로 암기까지 한 흐름으로 연결한다.

## 대상 사용자

- 영어 교재를 사진으로 빠르게 개인 단어장으로 옮기고 싶은 중·고등학생과 수험생
- 토익 빈출 어휘를 교재 단위로 정리하는 취업 준비생과 직장인
- 간호·의학 영어의 철자와 핵심 의미를 반복 학습하는 간호학생·보건계열 학습자
- 이미지 연상과 짧은 플래시카드 복습이 잘 맞는 모바일 중심 학습자

## 핵심 가치

1. **입력 시간 단축:** 사진 한 장에서 후보 단어를 만들고 일괄 교정한다.
2. **사용자 통제:** 오인식, 고유명사, 이미 아는 단어를 저장 전에 수정·삭제할 수 있다.
3. **기억 연결:** 뜻·품사·발음·예문·연상 설명과 출처가 있는 이미지를 한 카드에 묶는다.
4. **꾸준한 복습:** 오늘 복습할 양을 보여 주고, 응답 품질에 따라 다음 일정을 조정한다.
5. **오프라인 회복력:** 촬영 후 편집·학습 데이터는 네트워크 상태와 무관하게 로컬에 남는다.

## MVP 범위

- iPhone에서 카메라 촬영 또는 앨범 이미지 1장 선택
- 이미지 전처리 후 영어 단어 후보 OCR, 진행/실패/재시도 상태 표시
- 후보의 수정·삭제·추가, 중복 및 비단어 필터, 최종 사용자 확정
- 단어장 생성과 단어 저장
- 단어별 대표 뜻, 품사, 발음 표기/재생 가능 URL, 짧은 예문, 한국어 연상 설명
- 무료 이미지 검색 API 결과 선택 및 출처 표시
- 검색 결과가 부적합할 때만 사용자가 요청하는 AI 이미지 생성 fallback
- 앞/뒤 플래시카드, `다시/어려움/보통/쉬움` 평가, 다음 복습일 계산
- 로컬 우선 저장, 앱 재실행 후 복원, 사진 및 단어 삭제
- 일반·수능·토익·간호·의학 중 단어장 카테고리 선택

## MVP 제외 범위

- 계정, 다중 기기 실시간 동기화, 공유 단어장, 소셜 기능
- PDF 다중 페이지, 동영상, 손글씨 OCR, 문장 전체 번역
- 완전 자동 단어 확정 또는 의료 지식의 임상 검증/진단 기능
- 유료 결제, 광고, 웹 관리자, 교사용 학급 관리
- 고급 문법 분석, 작문 첨삭, 말하기 점수화
- AI 이미지 자동 생성 및 무제한 생성

## 핵심 사용자 흐름

1. 홈에서 새 단어장을 만들고 학습 분야를 선택한다.
2. 카메라 촬영 또는 앨범 선택 전에 사진 처리와 보관 방식을 안내하고 권한을 요청한다.
3. 이미지를 로컬 작업으로 등록하고 OCR 진행 상태를 표시한다.
4. 추출된 후보를 원문 위치와 함께 보여 주며 사용자가 수정·삭제·추가한다.
5. 사용자가 확정하면 단어 정보를 조회한다. 실패한 필드는 비워 두고 재시도/직접 입력을 허용한다.
6. 이미지 검색 결과와 출처를 보여 주고 사용자가 선택한다. 적합한 결과가 없을 때만 AI 생성 버튼을 제공한다.
7. 완성된 단어를 로컬 단어장에 저장하고 즉시 학습하거나 나중에 시작한다.
8. 오늘의 복습에서 카드를 뒤집고 난이도를 평가한다. 결과는 즉시 로컬 일정에 반영된다.

예외 흐름에는 권한 거부, 촬영 취소, 네트워크 단절, OCR 결과 없음, 중복 단어, 외부 API 부분 실패, 이미지 라이선스 누락을 포함한다.

## 성공 기준

MVP 베타에서 다음을 측정한다.

- 권한을 받은 사용자의 촬영/선택 시작 대비 교정 화면 도달률 90% 이상
- 일반적인 단어 페이지 1장 기준, OCR 시작부터 편집 가능한 후보 표시까지 중앙값 10초 이내(네트워크 OCR이면 연결 정상 조건)
- 저장된 OCR 후보 중 사용자가 수정 또는 삭제한 비율을 추적하고, 대표 테스트 세트에서 확정 단어 정밀도 95% 이상
- 신규 사용자의 첫 단어장에 5개 이상 저장 완료율 70% 이상
- 복습 알림/진입 사용자 중 당일 예정 카드 완료율 60% 이상
- 앱 강제 종료·오프라인 후에도 확정 단어와 복습 기록 유실 0건
- 이미지가 있는 카드의 출처 메타데이터 보유율 100%

지표 이벤트에는 사진 원본, OCR 전체 텍스트, 의학 학습 내용 등 개인정보성 payload를 넣지 않는다.
