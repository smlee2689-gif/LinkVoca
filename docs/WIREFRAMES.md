# 저해상도 와이어프레임

## 표기

- `[버튼]`, `(선택)`, `[ ] checkbox`, `⋮ 더보기`, `⌂ 하단 탭`으로 표현한다.
- 실제 구현에서는 모든 터치 대상 44×44pt, safe area, Dynamic Type을 적용한다.
- 와이어프레임은 콘텐츠 우선순위용이며 픽셀 단위 시안이 아니다.

## 플래시카드

```text
┌─────────────────────────────┐
│ ×  토익 800 핵심       3/20 │
│ ─────────●──────────────    │
│                             │
│ ┌─────────────────────────┐ │
│ │                         │ │
│ │       administer        │ │
│ │       /ədˈmɪnɪstər/     │ │
│ │                         │ │
│ │      [연상 이미지]      │ │
│ │                         │ │
│ │       탭해서 뒤집기     │ │
│ └─────────────────────────┘ │
│                             │
├─────────────────────────────┤
│          [답 보기]          │
└─────────────────────────────┘
```

뒤면은 카드 내부에 뜻→품사→예문→연상→이미지 출처 순으로 표시하며, 길면 카드 안에서 스크롤한다.

└─────────────────────────────┘
```

│                             │

│                             │

```
